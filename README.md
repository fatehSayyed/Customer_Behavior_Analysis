# 🛍️ Customer Shopping Analytics

## 📊 Project Overview

This project analyzes **customer shopping behavior** to identify purchasing patterns, customer trends, and product performance. The goal is to transform raw shopping data into meaningful **business insights and visual analytics**.

## 🛠️ Tools Used

* **Python (Pandas, NumPy)** – Data cleaning and preprocessing
* **SQL** – Data analysis and querying
* **Power BI** – Interactive dashboard and data visualization
* **PowerPoint / Report** – Presentation of insights and findings

## 🔄 Project Workflow

1. **Data Cleaning** using Python
2. **Data Analysis** using SQL queries
3. **Dashboard Creation** using Power BI
4. **Report Writing** to summarize insights
5. **Presentation** of results using PowerPoint
